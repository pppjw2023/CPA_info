The file "注册会计师名单.txt" contains the names of signing certified public accountants from all annual audit reports.

The code file "注册会计师个人信息抓取201710运行.py" performs the following functions:
1. Reads the list of certified public accountants.
2. Launches the Chrome browser using Selenium, simulates mouse clicks to access the search bar, enters the accountant's name, and clicks search.
3. Extracts the personal information displayed on the webpage using Xpath and writes it to a text file.
